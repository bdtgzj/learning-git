一. 什么是编程？（What is programming?）

  1. 【编程】即编写程序，让计算机听你话，帮你做事情、解决问题等。
  2. 【如何培养科学家、工程师？】STEM 教育就是专门来培养科学家和工程师的。

二. 为什么要编程？（Why program?）

  1. 【Science（科学）】探索、总结自然现象规律，形成一套知识体系。如：物理学、化学、天文学等。达尔文说：“科学就是整理事实，从中发现规律，做出结论。”
  2. 【Technology（技术）】是技术、技能、方法和过程的总和，用于商品、服务的生产或目标的实现。如造纸术、电子信息技术、生物技术等。
  3. 【Engineering（工程学）】应用科学、数学、技术等知识，创造产品、解决方案等，来解决问题。如：化学工程、电气工程、机械工程等。
  4. 【Mathematics（数学）】研究数量、结构、空间、变化以及信息等概念的一门学科。如：代数、几何、数论等。

二. 什么是 STEM？（What is STEM?）

  1. 【STEM】利用工程学（Engineering）方法，应用数学（Mathmatics）和科学（Science）知识，创造技术（Technology）和解决方案（Solution），来解决现实世界中的问题。
  2. 【历史】20 世纪 90 年代，由美国国家科学基金会提出，旨在提高科技发展的竞争力。美国政府从利益、安全出发，藉由 STEM 教育培育人才，促进经济以及科技的发展，维持全球领先，同时提高人民的科学素养。
  3. 【VEX IQ 平台】通过机器人课程，让学生动手探索 STEM，培养学生的科学思维、工程思维、创造力等。

三. 工程学（Engineering）

  1. 【工程师（Engineer）】我们日常所见的建筑、结构、车子等，都是工程师们利用科学、数学、技术等知识创造出来的。
  2. 【化学工程（Chemical Engineering）】利用物理和生物科学，将原材料或化学物质转化为更有用的形式，以解决问题。
  3. 【土木工程（Civil Engineering）】对物理和自然环境进行设计、建造和维护，来解决问题。
  4. 【电子工程（Electrical Engineering）】利用电、电子和电磁来解决问题。
  5. 【机械工程（Mechanical Engineering）】设计、创造机器和机械系统，来解决问题。
  6. 【特定工程（Specialized Engineering）】由两种或多种类型的工程组合而成的一种全新工程。如：生物医学工程、机器人工程。

四. 什么是机器人学？

  1. 【什么是机器人学（Robotics）？】机器人学属于工程学，通过设计、建造、操作和应用机器人，来解决特定问题。
  2. 【什么是机器人（Robot）？】机器人是一个人造机器，能够代替人类完成特定任务，尤其是重复任务、危险任务等。
  3. 【机器人类型】遥控机器人（Teleoperated Robots）、自主机器人（Autonomous Robots）、混合机器人（Hybrid Robots）

五. 机器人的应用

  1. 【工业生产机器人】用来制造汽车、电脑等。
  2. 【生活服务机器人】用来扫地、拖地、修剪草坪等。
  3. 【警察机器人】协助警察执行危险任务，如拆炸弹、调查危险环境等。
  4. 【深海机器人】在海底执行任务，发现深海新物种等。
