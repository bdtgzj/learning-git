一. 什么是编程？（What is programming?）

  1. 【编程】在虚拟世界中，自由表达想法。
  2. 【比如】我要在虚拟世界中创造一只猫，猫会唱歌、会飞、会说话等。
  3. 【淘宝】在虚拟世界中，构造了一个超级商场，里面的商品应有尽有。
  4. 【腾讯】在虚拟世界中，构造了一个社交帝国，让人们联系的更紧密。
  5. 【百度】在虚拟世界中，构造了一个超级老师，有问题就可以问百度。
  6. 【抖音】在虚拟世界中，构造了一个娱乐场所，刷视频就上抖音。

二. 为什么要编程？（Why program?）

  1. 【提升孩子的创造力】在虚拟世界中，孩子可以天马行空地表达任何创意。
  2. 【提升孩子的逻辑思维能力】增强孩子做事的条理性；增强孩子的推理与判断、比较与分析、抽象与概括、归纳与演绎的能力；
  3. 【提升孩子的解决问题的能力】
  4. 【培养孩子学习兴趣】基于项目的“做中学”教学方法，使孩子学习更有热情。
  5. 【人工智能时代的需求】在未来充满机器人、AI 的世界里，掌握编程技能让孩子更从容。
  6. 【增加升学竞争力】参加编程比赛，助力升学。

二. 什么是 STEM？（What is STEM?）

  1. 【STEM】利用工程学（Engineering）方法，应用数学（Mathmatics）和科学（Science）知识，创造技术（Technology）和解决方案（Solution），来解决现实世界中的问题。
  2. 【历史】20 世纪 90 年代，由美国国家科学基金会提出，旨在提高科技发展的竞争力。美国政府从利益、安全出发，藉由 STEM 教育培育人才，促进经济以及科技的发展，维持全球领先，同时提高人民的科学素养。
  3. 【VEX IQ 平台】通过机器人课程，让学生动手探索 STEM，培养学生的科学思维、工程思维、创造力等。

三. 如何编程？（How to program?）

  1. 【如何编程】使用编程软件、编程语言，来编写程序。
  2. 【编程软件】用来输入程序的软件工具，如 Scratch 编程软件。-->
  3. 【编程语言】用来向计算机发出指令。分为人类理解的语言和计算机理解的语言。

四. 栗子

  1. 【表达】创建一只小猫，每天走路 100 步，提示主人生命在于运动！

五. 机器人的应用

  1. 【工业生产机器人】用来制造汽车、电脑等。
  2. 【生活服务机器人】用来扫地、拖地、修剪草坪等。
  3. 【警察机器人】协助警察执行危险任务，如拆炸弹、调查危险环境等。
  4. 【深海机器人】在海底执行任务，发现深海新物种等。
