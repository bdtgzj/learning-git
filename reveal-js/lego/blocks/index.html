一. 积木

1. 【什么是积木？】木头或塑料制成的固体玩具。
2. 【积木能用来做什么？】对现实世界进行建模，如搭建房子、车子、飞机、小动物等。
3. 【积木的作用是什么？】开发智力、锻炼动手能力和想象力。
4. 【什么是乐高体系？】在乐高体系中，积木被称为颗粒（或元件），使用一些零零碎碎的乐高颗粒，通过彼此间的连接，可以组装成更大的或者一系列的作品。
5. 【积木收集】更多的积木类型，更多的搭建方案。对于乐高爱好者来说，收集各种类型的颗粒也是乐趣之一。

二. 颗粒的尺寸

1. 【什么是颗粒的尺寸？】颗粒的长、宽、高。
2. 【什么是基准颗粒？】基准颗粒的长、宽、高分别称为一个标准单位，作为衡量其他颗粒尺寸的标准。乐高中的基准颗粒是 1x1 砖型颗粒，长:宽:高=x:5:6。
3. 【如何描述颗粒？】一般把尺寸小（宽）的数字放在尺寸大（长）的数字前面。
4. 【1 x 1 颗粒】一乘一颗粒，即基准颗粒，一个单位宽，一个单位长，一个单位高。
5. 【1 x 2 颗粒】一乘二颗粒，一个单位宽，两个单位长，一个单位高。
6. 【1 x N 颗粒】一乘恩颗粒，一个单位宽，N 个单位长，一个单位高，N 代表所有可能的长度。
7. 【2 x 4 颗粒】二乘四颗粒，两个单位宽，四个单位长，一个单位高。

三. 颗粒的连接

1. 【凸粒】大部分颗粒的顶部都会有凸粒，上面印有 LEGO 字样，凸粒方便我们识别出颗粒的宽和长。凸粒有大小高低之分。
2. 【管槽】大部分颗粒的内部都会有管槽，管槽有大小空心实心之分。
3. 【横档】大部分颗粒的内部都会有横档，横档分布在颗粒四周的壁上。
4. 【连接】把一个颗粒的凸粒插入另一个颗粒的管槽和横档之间的空间，便能把两个颗粒牢靠地连接起来。

四. 颗粒的形状

常见颗粒形状有：砖型颗粒、板型颗粒、斜面颗粒、拱形颗粒、瓦片颗粒、嵌板颗粒、圆柱颗粒、圆锥颗粒、板型圆柱颗粒、特殊颗粒、底板、装饰元件等。

1. 砖型颗粒

  • 【外观】一个长方体，与盖房子用的砖类似，是乐高体系中的“支柱颗粒”。
  • 【应用】可用于搭建建筑的墙体、汽车、飞机、人物、动物等，可用于装饰。
  • 【1x1 砖型颗粒】可用于搭建人物、马赛克、小型动物等。
  • 【1x2 和 1x3 砖型颗粒】常被拼成一列列的，用来支撑整体结构或仅仅用于装饰。
  • 【1xN 砖型颗粒】搭建墙壁，因为他们厚度合理，几乎是真实世界里建筑墙壁的翻版。
  • 【2x4 砖型颗粒】核心颗粒，在它的基础上能增加其他颗粒。

2. 板型颗粒

  • 【外观】一个长方体，同砖型颗粒，高度是砖型颗粒的三分之一。
  • 【应用】可用于调整高度、做内部支撑、按照真实比例缩放、微调、做视觉效果等。
  • 【1x1 板型颗粒】几乎在所有模型中，都能找到它的身影。如可用于搭建人物、马赛克、小型动物等。
  • 【1x2 和 1x3 板型颗粒】
  • 【1xN 板型颗粒】搭建直升机的螺旋桨、火车头两侧长长的彩色条纹。
  • 【2xN 板型颗粒】让你用最少的颗粒来完成作品。

3. 斜面颗粒

  • 【外观】有一个或几个从顶部贯穿到底部的斜面，斜面角度从 18 度到 75 度都有，最常见的是 33 度和 45 度斜面颗粒。
  • 【应用】可用于搭建屋顶、飞机的斜翼、常绿乔木等。

4. 特殊颗粒

  • 【外观】外观独一无二，不属于其他任何类别。
  • 【应用】在普通或特殊场合，都能派上大用场。
  • 【一分二颗粒】仅有一个凸粒的板型颗粒，也可称为中心有一个凸粒的瓦片颗粒。

5. 拱形颗粒

  • 【外观】外观像座桥。
  • 【应用】可用于搭建建筑门框、窗框等。

6. 瓦片颗粒

  • 【外观】看起来像没有任何凸粒的板型颗粒。
  • 【应用】

7. 嵌板颗粒

  • 【外观】可看作将几块瓦片颗粒以一定角度相连，形成的一面或两面壁较薄的颗粒。
  • 【应用】

8. 圆柱颗粒

  • 【外观】圆柱体外形。
  • 【应用】灯柱、建筑支撑体。

9. 圆锥颗粒

  • 【外观】圆锥体外形，看上去像倒置的蛋筒冰淇淋。
  • 【应用】树干、水枪喷嘴。

10. 板型圆柱颗粒

11. 底板

  • 【外观】“小的底板”类似“大的板型颗粒”，两者最大区别是高度不同，前者比后者薄很多。
  • 【应用】一个平台，承载作品，方便搬运。

12. 装饰元件

  • 【外观】窗户、门、树、栅栏、旗等。
  • 【应用】让模型更生动、逼真。

四. 颗粒的精度

1. 乐高颗粒精度非常高，不亚于飞机零部件的精度。
2. 颗粒精度即颗粒尺寸的精确度，包括颗粒的长度、宽度、高度、管槽、边缘、横档等。
3. 精度低的颗粒存在误差，误差的累计，会导致无法搭建稳定的模型。

五. 颗粒的颜色（色彩学）

1. 【乐高颗粒拥有丰富的颜色】黑色、白色、红色、黄色、蓝色、绿色、透明色、深绿色、栗色、淡蓝色、暗灰色、亮橙色、粉色等。
2. 【颜色让模型更逼真、更丰满】如用红色和黄色颗粒搭建消防车，那么看起来像真的一样。
3. 【用有限的颜色创造出新的色彩视觉效果】将白色颗粒放在黑色颗粒旁边，白色颗粒就会显得有些发灰；将黄色颗粒放在红色颗粒旁边，模型就会呈现出一点橙色。

六. 颗粒的几何学

1. 【1x1 基准颗粒的几何现象】宽:高=5:6，含义是：将 5 个 1x1 颗粒连起来，高度与 1 个 1x6 颗粒的长度相等。
2. 【管槽和凸粒的几何现象】管槽的内径=凸粒的直径，含义是：如何“一个颗粒的凸粒数”少于或等于“另一块颗粒的管槽数”，那么一个颗粒的凸粒可以直接插到另一个颗粒的管槽中。
3. 【板型颗粒和转型颗粒间的几何现象】板型颗粒的高度:砖型颗粒的高度=1:3，含义是：三块板型颗粒重叠起来，可以代替一块砖型颗粒，可用来做视觉效果。
