一. 搭建方法

1. 【什么是搭建方法？】将颗粒连接在一起的方法。
2. 【搭建方法有多少种？】假设你有 6 块 2x4 的颗粒，可以搭建出 102 981 500 种结果。
3. 【如何选择搭建方法？】视情况，选择最好的搭建方法。

二. 常用的 3 种搭建方法

1. 完全重叠。
2. 部分重叠。
3. 阶梯排列。

1. 完全重叠法

  1. 【什么是完全重叠法？】相同尺寸的颗粒，上下堆叠在一起。
  2. 【缺点】完全重叠的颗粒们，如果没有周围颗粒的支撑，是非常不坚固的。
  3. 【应用】一般出于美观的需要，而不是结构上的需要。如搭建彩色条纹外墙，给飞机的尾翼添加五颜六色的条纹。
  4. 【最佳实践】选择完全重叠法时，务必用长砖型颗粒或长板型颗粒分别从上面和下面固定好完全重叠的颗粒，避免“壳蛋效应”。
  5. 【壳蛋效应】指外观完好的蛋壳，表面上十分坚硬，却只要轻轻一捏就会变成一撮碎片。

2. 部分重叠法

  1. 【什么是部分重叠法？】颗粒间，部分堆叠连接在一起，就像造房子时砌墙一样。
  2. 【优点】部分重叠连接在一起的颗粒们，彼此间的相互作用力（也叫颗粒间互锁）最强，搭建出来的模型最坚固。
  3. 【应用】一般用于保证模型坚固。如搭建墙面、固定窗户等。

3. 阶梯排列法

  1. 【什么是阶梯排列法？】把“上层颗粒的边”沿其“下层颗粒的边”向内缩进，形成类似楼梯的外形。
  2. 【应用】搭建屋顶、雕塑、像素画等。
  3. 【最佳实践】使用阶梯排列法时，曾与层之间的连接使用部分重叠法，使模型更坚固。

三. 搭建墙壁

  1. 【搭建一面墙壁】用“部分重叠法”可以搭建出一面坚固的墙壁。
  2. 【搭建两面墙壁】不要企图把两面分别搭建的墙连接起来，达到强强联合的目的，而是要同时搭建两面墙，并使用“部分重叠法”，把两面墙互相连接起来，彼此借助对方的力量，结构才坚固。
  3. 【部分重叠法】使模型坚固的关键。

五. 支撑

  1. 【什么是支撑？】用长颗粒、梁、柱使模型结构更坚固。
  2. 【为什么需要支撑？】小的模型可能不需要任何支撑，而中大模型需要很多支撑，以保证模型不会倒塌。
  3. 【把模型内部填满，不是更坚固吗？】是更坚固，但比较适合搭建小模型，对于大模型来说，会使模型变得很重，会消耗很多不必要的颗粒。
  4. 【支撑主要起功能性作用，一般不要求看起来美观】因为支撑一般是隐藏在模型内部或后面的，可以使用任何颜色的颗粒来搭建支撑。
  5. 【应用】当模型需要手提、搬运、展示时，支撑使模型更坚固。
  6. 【栗子】用梁柱连接、支撑建筑墙壁，使建筑更坚固。

六. 梁

  1. 【什么是梁？】梁是水平排列的颗粒，它们与纵向的颗粒联合起来，使结构更坚固。
  2. 【简易梁】一个颗粒就可以是一个梁，如 2x8 砖型颗粒就是一种简易梁。
  3. 【组合梁】相当于一个子结构，通过砖型颗粒和板型颗粒搭建出一个组合梁。

七. 柱

  1. 【什么是柱？】柱是垂直排列的颗粒，它们与横向的颗粒联合起来，使结构更坚固。
  2. 【柱的结构类型】简单柱、复合柱、烟囱柱、锁眼柱、混合柱。

1. 简单柱

  1. 【什么是简单柱？】使用完全重叠法，把同一尺寸（2x2 或 2x3 或 2x4 等）的砖型颗粒堆叠在一起，形成最简单的柱。
  2. 【优点】搭建简单。
  3. 【缺点】搭建的越高，结构稳定性越差。
  4. 【栗子】使用 2x2 或 2x3 或 2x4 的砖型颗粒堆叠出简单柱。

2. 复合柱

  1. 【什么是复合柱？】使用部分重叠法，堆叠砖型颗粒，形成复合柱。
  2. 【优点】结构稳固。
  3. 【缺点】使用的颗粒较多，自身重量太重。
  4. 【栗子】每层使用 2 个 2x4 砖型颗粒，使用部分重叠法堆叠出复合柱。

3. 烟囱柱

  1. 【什么是烟囱型？】中空的复合柱。
  2. 【优点】结构稳固，使用的颗粒较少，自身重量较轻。
  3. 【缺点】要检查中空部分是否保持着正方形。可以搭建一个 2x2 简单柱来做检查。
  4. 【栗子】每层使用 1x4 和 1x2 砖型颗粒，使用部分重叠法堆叠出烟囱柱。

4. 锁眼柱

  1. 【什么是锁眼柱？】使用部分重叠法，堆叠 1x2 （或 1x3） 砖型颗粒，使中空部分形如锁眼，形成锁眼柱。
  2. 【优点】同烟囱型结构，比烟囱型结构更美观。
  3. 【缺点】由于由太多颗粒组成，当提起时，可能会弯曲、扭曲。
  4. 【栗子】每层使用 1x2 砖型颗粒，使用部分重叠法堆叠出锁眼柱。

5. 混合柱

  1. 【什么是混合柱？】同时使用“复合柱”和“烟囱柱”的搭建方法，形成混合柱。
  2. 【优点】兼具“复合柱”和“烟囱柱”的优点。
  3. 【栗子】搭建烟囱柱时，每隔 2 层，添加一层复合柱。

六. 模型拆解

  1. 【什么是模型拆解？】把搭建好的模型，还原成颗粒。
  2. 【为什么要拆解模型？】模型已经展示过，接下来搭建新模型需要用到模型上的颗粒，所以需要拆解模型。
  3. 【拆解模型方法】模型拆成大模块，大模块拆成小模块，小模块拆成颗粒。
  4. 【拆解问题】有些颗粒彼此连接得太紧，很难分开，如两个 1x2 小板型颗粒之间，连指甲都塞不进，这时怎么办？
  5. 【使用颗粒分离器拆解】使用两个颗粒分离器，用凹槽和凸粒分别连接到 2 个板型颗粒上，按压颗粒分离器使颗粒分离。
  6. 【使用砖型颗粒拆解】使用两个砖型颗粒，分别连接到 2 个板型颗粒上，掰动砖型颗粒，使板型颗粒分开。

六. 搭建法则

  1. 【大处着眼，小处着手】把模型划分成小模块，使搭建更容易，使工程不那么让人望而生畏。如搭建摩天大楼，可以分别搭建其中的几层，然后把它们连接起来；搭建小货车时，先搭建驾驶室，再考虑车厢部分。
  2. 【选择合适的连接方式】取决于要搭建的模型，虽然部分重叠法很好，但也不总是适用于所有情况。
