制作教室助手

（Make-A-Classromm-Helper）

schedule45-90 分钟 stars初级（Beginner） face3-5 年级

一. 场景（Connect）

同学们，想一想我们的教室中有哪些“助手”帮助着我们更好的学习呢？

一. 场景（Connect）

  1. 书架（Book Stand）帮助我们整理书本。
  2. 铅笔（Pencil）帮助我们写字。
  3. 橡皮（Eraser）帮助我们擦字。
  4. 尺子（Ruler）帮助我们测量长度。
  5. 笔盒（Pencil Box）帮助我们整理铅笔。
  6. 再想一想，我们的学习生活中，是否存在问题，可以通过“助手”来解决呢？

二. 发现问题（Find a Problem）

大家一起讨论、发现问题、确定问题、并记录问题。

三. 头脑风暴（Brainstorm）

  1. 花 3 分钟时间，独自头脑风暴，尽可能多地产生解决问题的想法。
  2. 想法可以用笔画出来，想法可以用积木搭建出来。
  3. 小组内轮流分享想法，并选择最佳的、可用积木实现的想法。
  4. 每个小组的想法都是个性化的，不一样的。
  5. 分享你的想法、思考。

四. 选出最好的想法（Choose the Best Idea）

依据设计标准（Design Criteria），选出最好的想法。

五. 制作（Make）

  1. 使用积木实现你的想法。
  2. 在制作过程中，测试和分析你的想法，必要时可进行改进。

六. 评估（Evaluate）

  1. 依据设计标准，测试和评估模型。

七. 展示（Present）

  1. 向大家展示你的模型，分享你的想法、草图（Sketche）。
  2. 把模型放到展示台上，来张合影吧！

八. 整理（Tidy Up）

同学们今天非常棒，最后让我们拆解模型，把积木分类存放到收纳盒吧。
